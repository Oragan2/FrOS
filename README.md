# MeOS
A custom operating system and kernel made for fun in 64-bits from scrathch

Current feature
| Feature | Status |
| ------- | ------ |
| Screen | ✅ |
| Basic keyboard | ✅ |
| File systeme | 🟠 |
| Processes | ❌ |
| Interuptions | ✅ |
| PCI | 🟠 |
| Memory | ❌ |
(Probably missed some)

Currently working on making memory pagging

To compile the OS you'll need : 
- i686-elf-gcc
- i686-elf-ld
- nasm
Then run 'make', if you have qemu you can run 'make run' to launch the OS

It also contains a python file made to initialize a File system for the OS
